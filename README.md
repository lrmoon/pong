## Game Title: 

Pong

## Description:

This is a 2d ping pong game. I chose to make pong because I hate css.

## Directions:

Player 1 paddle controls:
 W (up) 
 S(down)

Player 2 paddle controls:
ArrowUp (up)
ArrowDown (down)
